A Survey on Active Learning and Human-in-the-Loop Deep Learning for Medical Image Analysis

Abstract

    关注四个关键领域：
    （1）主动学习：选择最佳的数据来进标注，以获得最佳的性能；
    （2）与模型的输出进行交互：使用迭代反馈来引导模型优化给定的预测，提供有效的方法来对预测进行反馈；
    （3）实际考虑：开发和部署需要考虑的关键问题；
    （4）未来前景和未解决的问题
    医学成像正受益于分析图像等结构化数据的新计算技术的发展。图像采集、分析和解释算法的发展正在推动创新，特别是在重建、跟踪、分割和建模等领域。

Introduction

    医学图像很难解释，需要事先的专业知识来理解。数据所需的经验和先验知识意味着在标注医疗数据时，往往存在观察者间和观察者内部的可变性。因此，在标注时，需要许多oracles标注并达成共识。
    深度学习处理医疗图像在实际应用中，往往存在以下问题：复杂医学数据、法规遵从性问题和是否信任模型。
    在临床中，医学图像处理存在三个挑战：
    （1）缺乏训练数据；
    （2）最终训练结果：虽然DL效果很好，但是在关键安全问题上，即使是最小的错误也会导致灾难性的结果，因此，得到可信的输出需要对预测结果进行交互式解释，即用户必须有能力纠正和覆盖自动预测的结果；
    （3）透明度和可解释性：目前，大多数DL模型是黑盒模型，用户在解释、理解或纠正模型如何进行预测的有效方法上是有限的。

Active Learning

Query Types

    Stream-based Selective Sampling：传入未标注的数据点的连续流
    Membership Query Synthesis：假设我们不是从真实的数据点分布中提取，而是生成一个需要标注的数据；模型可能不知道分布中不可见的区域，因此无法生成这些区域的数据；目前，基于GAN展示出了巨大的前景。
    Pool-based Sampling：存在一个大的未标注的真实数据集，从中选择样本进行标注；这种方法计算代价可能很高，因为每个迭代都需要对分布中的每个数据点进行度量评估；然而，却是与DL方法结合最有前景的，因为DL方法本质上依赖于批处理。

Evaluating Informativeness

Uncertainty

    一个数据的预测结果越不确定，我们可以通过在训练集中包含该样本的真值中获得的信息就越多（least confident sampling）。
The Power of Ensembles for Active Learning in Image Classification

Representativeness

    不确定性有可能只关注小区域的分布，而训练样本从同一区域的分布将引入冗余的选择策略，或可能倾斜模型对一个特定区域的分布。增加代表性度量旨在鼓励选择策略从分布的不同区域进行抽样，并增加样本的多样性.

Generative Adversarial Networks for Informativeness

Fine-tuning vs Retraining

    AL的最后一步是使用新获得的标注数据来改进模型。两种方式：使用所有可用数据对模型进行重新训练，包括新获得的标注数据；只使用新标注的或者新标注+现有的子集对模型进行微调。

The Final Percent: Interactive refinement of model outputs

    之前的介绍中，人标注数据用于训练模型。在模型训练中，需要一个human-in-the-loop解释模型预测结果，并可能对其进行改进，以获得对未知数据最准确的结果。     用户将预测引导到更准确的结果或纠正错误预测，使用户能够在与模型输出交互时采取最佳行动，并减轻人为不确定性。

Interactive Refinement

    允许用户改进模型输出，并将更准确的结果反馈给模型以进行改进。

Practical Considerations

Noisy Oracles

    医学图像数据的标注是通过聚合来自多个专家的标注获得的；如果我们从没有领域专家中获得标签，以及在使用非专家标注时，我们可以使用什么技术来减轻标准质量的可疑下降，以避免任何潜在的准确性损失。

Weakly Supervised Learning

    我们可以使用弱标注数据开发高性能模型，而且由于弱标注需要更少的专业知识，它们可以更快地从非专家群体中获得，而且比标准标注的准确性损失更小。
